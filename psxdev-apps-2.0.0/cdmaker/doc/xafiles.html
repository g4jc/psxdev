XA file settings

Like the exclusion list for unwanted files, there is an inclusion list for files to be treated as XA files. XA files are using the complete 2336 bytes/sector, while normal data files are using only 2048 bytes of the 2336 bytes of a sector (and the remaining bytes are used for subheaders, ECC and EDC informations)
