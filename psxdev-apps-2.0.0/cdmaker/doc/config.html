PSXDEV CD Maker Configuration

Nearly everything is configurable.

CDRDAO command

Enter the name of your CDRDAO command here. If CDRDAO is not in your path, enter it with the complete path here.

MKISOFS command

Enter the name of your MKISOFS command here. PSXDEV CD Maker uses a custom version of mkisofs, called psx-mkisofs. That version writes images based on 2336 bytes/sector. If this command is not in your path, please enter it with the complete path.

DISKUSAGE command

This command is internally used to determine the used disk space. The first printed word of this commands output should be the amount of used megabytes.

SHELL command

The complete path to a shell. Currently not used.

ISO image filename

This is the filename (without path!) of the ISO image to be created

TOC sheet filename

This is the filename of the TOC sheet to be created. Please do not enter a path, since this argument is appended to the output path.
