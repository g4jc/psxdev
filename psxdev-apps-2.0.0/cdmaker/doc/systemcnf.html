Generating the SYSTEM.CNF;1 file

Normally you won't need this feature and want to provide your own and self edited SYSTEM.CNF file. However, this is an important file and if it's missing or corrupt your master would be unusable. So you better let psxcdmaker create a correct one!

Click here to turn on this feature!

TCB

The number of task control blocks initially allocated. [set default]

EVENT

The maximum number of system events which can be used. [set default]

STACK

The initial stack pointer value. [set default]

BOOT

This file will be booted. The path must be of the following format or the load will be rejected:

LICENSE

The content of this file will be written to the system area of the disc.

EXCLUSION LIST

This is a list of shell patterns. All files which will match against one of these patterns are excluded from the CD-ROM! This is very useful if your master directory does contain temporary files like backups, secret documentation and so on.
