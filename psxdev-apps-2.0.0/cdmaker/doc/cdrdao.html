CD-Writer settings

If you want to write a CD as a normal user, make sure you have the right to access the CD writer. This can be done in several ways:
  • run PSXDEV CD Maker as root (not recommended!)
  • set the SUID bit on the CDRDAO executable
  • give ownership of the CD writer device node to a new group (i.e. cdwriter) and add yourself to this group (recommened!)

Device

This is the SCSI device to use. Please refer to the CDRDAO documentation for further details.

Driver

It is normally not required to select a driver here, if you leave this field empty the correct driver should be automatically detected. However you may want to specify driver specific options. Please refer to the CDRDAO documentation for more details.

Speed

Select the CD writer speed. 0 is treated as maximum possible speed.

Simulate

Do everything with the laser turned off. Recommended!

Eject

Eject the medium after process completion.
