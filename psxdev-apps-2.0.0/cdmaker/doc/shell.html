Shell input/output

Shell Window

All subprocesses of PSXDEV CD Maker are real subprocesses and are running concurrently to the main application. All their shell output is collected in this terminal emulation. You have a huge scrollback buffer to analyse the command output and the possibility to interact with a command (like pressing CTRL-C to abort it!).

Process management

Once the Master or Write button was pressed, the shell is busy processing a shell command (mkisofs or cdrdao). During this time you can still work with the complete application, but the above mentioned buttons are ghosted as long a shell process is working. If you launched a process by mistake, simply press CTRL-C to abort it! If a shell process is still running while you are quitting the application, the subprocess is immediately killed.
