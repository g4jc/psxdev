Tutorial

This small tutorial should help you understanding this application.

Preparation of your input files

PSXDEV CD Maker uses a master directory to create an ISO image from it, so you have to populate a directory with files. All these files should already fulfill some limitations, which are
  • all filenames must be lowercase
  • all filenames must be 8+3 (filename.dat)
  • a maximum of 30 files per directory (including directories)
  • a maximum of 45 directories per disc

Mastering an image

If your master directory is filled with files, ensure they are fulfilling the above limitations (there is currently no check option in this application). Fill out all the register cards regarding the image, which are Paths, ISO9660, PSX and Audio. Make sure your input path points to your master directory and that it does not use more than 660MB (use the disk usage button). Make sure your output path points to a user writable directory with at least 660MB free disk space (look at the gauge). Now press the Master button and analyse the process output. If you find error messages here you have to correct them first before proceeding to the write process!

Writing an image

The master process did not waste your CD recordables, but this process can. So you should always test this process with the simulate option! Please study the CDRDAO documentation to understand all the options in the CD-Writer register card. Press the Write button to start the write process.
