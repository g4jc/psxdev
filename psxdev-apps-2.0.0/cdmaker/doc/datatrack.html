The Data Track

ISO 9660 filesystem

Refer to an ISO 9660 filesystem documentation first, then fill out the fields. It is important that the application ID and the system ID are uppercase PLAYSTATION.

Information files

These files are completely optional and are rarely used. You can specify some files on the volume to be created, please refer to an ISO 9660 documentation for more details.

Miscellaneous

The only currently supported option is Follow Symbolic Links. If this option is set all links are followed and the original content is written. So you could soft link all files into one directory with this option and the real files will be written. If not set all links will be ignored.

All inactive options are currently disabled, because they would confuse the user and could lead to incorrect mastered images if set by mistake.
