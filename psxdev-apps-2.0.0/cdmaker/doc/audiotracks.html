Setting up the directory paths

Input path

The input path points to the directory which holds the content for the CD-ROM to be created. All files in this directory are written to the root directory of the new CD-ROM. If you want to exclude files you can do that with the exclude pattern in the Data Track Section.

Checking the disk usage

To determine the disk space used by a selected directory, just click the Disk Usage button. The calculated size will be displayed relative to a virtual 700MB CD-ROM in the fuel gauge near the button.

Output path

This path points to a directory with at least as much free space as an ISO image would need (typically ~700MB). The temporary TOC file is written to this directory and the image file itself. If you are using the on the fly mode you need only a few KB for the TOC file as no data images are created (data is piped directly to the CD writer).
