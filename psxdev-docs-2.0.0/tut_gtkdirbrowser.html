Using the GtkDirBrowser widget

The GtkDirBrowser widget


This widget is part of the psxdev-apps-devel package.

You will also find this example there, including a Makefile. You will need the following requirements to proceed:
  • Gtk+ 1.2 or later
  • C compiler
  • POSIX compatible operating system
It does not matter if you're testing this with LINUX, SOLARIS, WINDOWS; it should work on all platforms supported by GTK+.


click on image to enlarge screenshot


GtkDirBrowser features

  • Parent class is GtkCList
  • Customizable Roots (starting points)
  • Emits a signal on path_changed
