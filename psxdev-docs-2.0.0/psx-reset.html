psx-reset - reset the PSX Daniel Balster/psxdev-tools
  • SYNOPSIS OF psx-reset


  • DESCRIPTION OF psx-reset
    
psx-reset does a software reset of the PSX.



  • NOTES ON psx-reset
    
This is only available if the communication link is still working; if the PSX is crashed you have to do the RESET manually by pressing the appropriate button.
