PSXDEV and the MDEC chip

The MDEC chip is responsible for fast decoding of compressed images. The standard image format on the PSX is BS. PSXDEV can encode and decode these images on the host using the libbs.so shared library, the BS gimp plug-in or with the shell utilties.
