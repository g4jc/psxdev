Question: How does PSXDEV communicate with the PlayStation?

PSXDEV uses a combination of software and hardware to establish a network link between a PC and a PlayStation. You need the following requisites:
  • An i386-based, IBM compatible PC with at least one emtpy ISA slot
  • A PC commslink ISA card (try www.mountstuffmore.de)
  • An ActionReplay/GameBuster cheat cartridge (try www.mountstuffmore.de)
  • A replacement firmware called CAETLA
This hardware components will allow you to establish a parallel communication link between the desktop computer and the video game console.
