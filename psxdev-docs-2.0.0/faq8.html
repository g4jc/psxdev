Question: What are the minimum requirements to use PSXDEV?

Any x86 compatible processor will work. However, the binary distribution is optimized for at least an i586 class processor, but you can recompile it for your processor. Memory usage is quite low: the kernel driver uses less than 32KB and all tools are using shared libraries. A full installation requires about 30MB of your harddisk space.
