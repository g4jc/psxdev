  • SYNOPSIS OF mc-up


  • DESCRIPTION OF mc-up
    
With mc-up you can transfer local files to a PSX memory card. You can use shell wildcards to specify which files should be sended. You can upload everything you want, but this makes no sense.




  • BUGS OF mc-up
    
Slots are numbered 0 and 1 instead of 1 and 2. This is a design bug in all memory card tools but it won't be changed anymore.
