PSXDEV CD Maker has an integrated html based online help including a wizard like tutorial. I am doing my best to write the best documentation possible. Maybe I will integrate this widget in all my applications.
This is the path tab. Input and output paths are selected here, and you can do disk space checks (used space for input, free space for output).
These are the settings for the base ISO 9660 filesystem.
Here are some PSXDEV special filesystem attributes. The XA inclusion list lets you specify patterns, and all files in the filesystem which will match against such a pattern will be written in XA mode (2336 bytes/sector). So you do not need to single click each XA file each time you are creating a new layout, a simple pattern like *.xa is sufficient.
These are the settings of the CD writer application which is used internally.
During all sub processes, a full featured shell terminal lets you interact (if necessary; lets say CTRL-C or "press return") with the processes. psx-mkisofs and cdrdao are started as sub processes
You can really configure everything. Even the GUI will be configurable with an XML file!
