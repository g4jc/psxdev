set PSX frame buffer mode

psx-fbmode - set PSX frame buffer mode Daniel Balster/psxdev-tools
  • SYNOPSIS OF psx-fbmode


  • DESCRIPTION OF psx-fbmode
    
With psx-fbmode you can control the PSX frame buffer display mode if you are in CAETLA main menu. This is very useful if you are using a script to display images on the remote machine.
