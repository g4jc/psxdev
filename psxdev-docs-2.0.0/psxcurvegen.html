imagedescription
This was written in one or two hours. It's just an GAMMA control widget abused for free-form (or spline controlled) curve editing. You can paint a curve, specify its range and datatype and export it as raw data. That's all. It has an experimental feature call live update: if you know the absolute address of a curve in the remote memory, you can directly upload it into a running application!
