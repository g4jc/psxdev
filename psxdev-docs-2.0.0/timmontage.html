  • DESCRIPTION OF timmontage
    
timmontage will merge many TIM images into one new TIM image. This is mainly used by me to create multi-palette TIM images



  • EXAMPLES FOR timmontage
    
Take a look at the graphics tutorial



  • BUGS OF timmontage
