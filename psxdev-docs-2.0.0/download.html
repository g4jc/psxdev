Download area

Important! You only need to download the binary RPMS, the source codes are only needed if you want to recompile everything or just want to take a quick view into the sources. All binaries were compiled for at least an i586-type processor and with all optimizations turned on. Please take a look at the installation guide where you will get more informations.

TIP: Instead of single clicking all the links you should use a recursive web sucker. I recommend to use an utility like wget, download it and type in at the command line prompt

wget -np -nv -r http://psxdev.de/download
To ensure that the downloaded files are not corrupted you should cross check them with the MD5 checksums in the last box on this page; simply type 
md5sum *
 in your download directory. You could also use the rpm --checksig and tar --test functionality.

TIP: You don't need the SOURCE tarballs and the patches if you have downloaded the SRPM packages. You can extract the sources, patches and spec files by typing

rpm2cpio filename.rpm | cpio --extract --make-directories
The SRPM packages have the advantage that they include the SPEC file, which contains detailed building instructions (for RPM).

TIP: Do not download the ADMIN and ELF packages, you don't really need them.
