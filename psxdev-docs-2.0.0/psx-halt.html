psx-halt - get control over the PSX Daniel Balster/psxdev-tools
  • SYNOPSIS OF psx-halt


  • DESCRIPTION OF psx-halt
    
psx-halt stops the execution of a currently running application on the PSX immediately. You can use psx-resume to resume the execution.



  • EXAMPLES FOR psx-halt
    

    psx-halt
    will stop the current application from running on the PSX
